# shinka-game (しんかゲーム)

端末を傾けて落下する円を合体させる、シンプルな物理演算ベースのウェブゲームです。同じサイズと色の円が触れ合うと、より大きな1つの円に進化（しんか）します。

## デモ

ゲームはこちらで遊べます: **https://github.com/code4fukui/shinka-game

## 遊び方

1.  モバイル端末で[デモリンク](https://code4fukui.github.io/shinka-game/)を開きます。
2.  「しんかゲーム」ボタンをタップしてモーションセンサーのアクセスを許可し、ゲームを開始します。
3.  端末を傾けて重力を操作し、円を動かします。
4.  同じサイズの円同士が触れ合うように誘導します。自動的に合体してより大きな円になります。

## 特徴

-   [eg2d.js](https://github.com/code4fukui/eg2d/) ライブラリを介した [Matter.js](https://brm.io/matter-js/) による物理演算ベースのゲームプレイ。
-   同じサイズの円が自動的に合体し、段階的に大きな円を作成。
-   端末のモーションセンサーを使用したインタラクティブな重力制御。
-   インストール不要で、すべてブラウザ上で動作。

## 依存ライブラリ

-   [eg2d.js](https://github.com/code4fukui/eg2d/)
-   [rnd.js](https://js.sabae.cc/rnd.js)

## ローカルでの実行方法

1.  このリポジトリをクローンします。
2.  モダンブラウザで `index.html` ファイルを開きます。
3.  開始ボタンをクリックして始めます。重力制御による完全な体験を得るには、モバイルブラウザを使用してください。
